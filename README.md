# Hi, I'm Vojta

I am a student of Open Informatics at CTU in Prague with a strong interest in artificial intelligence, software engineering, and real-world technical projects.

My main interest in AI is not just using new tools, but understanding how they work, how to build with them, and where they are genuinely useful in practice. I am especially interested in modern AI workflows around LLMs, automation, and agentic systems, but I also care about the broader technical foundations behind intelligent systems.
